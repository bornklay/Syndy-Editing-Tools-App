# Demucs Audio & Video Toolkit

Demucs Audio & Video Toolkit is a desktop application that allows users
to process audio and video files using AI and multimedia tools.

The application provides music source separation, video conversion,
and audio extraction features in a simple graphical interface.

## Installation

This software is distributed as a Windows installer.

Run the installer and follow the setup instructions to install the
application on your system.

## First Launch

The first time you use the vocal separation feature, the application
will automatically download the required AI model for Demucs.

An internet connection is required for this initial download.

After the model has been downloaded successfully, the application can
run completely offline and does not require internet access.

## Features

• AI Vocal Separation
Separate vocals and instruments from music using the Demucs AI model.

• Multi-Stem Extraction
Extract individual stems such as:

* Vocals
* Drums
* Bass
* Other instruments

• Video Format Conversion
Convert video files into different formats using FFmpeg.

• Video to Audio Extraction
Extract audio tracks from video files and save them as audio formats
such as WAV, MP3, or other supported formats.

• Local Processing
All audio and video processing is performed locally on your computer.
